Metrics enable optimization. Comprehensive frameworks reveal all opportunities. The firms that measure effectively improve continuously.

## The Input Metrics

Input metrics measure volume generation.

> Signals identified per period.
> Outreach campaigns executed.
> Cost per signal generated.
> Source effectiveness by channel.

The metrics reveal generation efficiency.

## The Throughput Metrics

Throughput metrics measure conversion.

* Contact rate from outreach.
* Meeting rate from contact.
* Offer rate from meeting.
* Close rate from offer.

The metrics reveal funnel efficiency.

## The Output Metrics

Output metrics measure results.

+ Deals closed per period.
+ Total value closed.
+ Average deal size.
+ Return on sourcing investment.

The metrics reveal overall performance.

## The Optimization Metrics

Optimization metrics reveal improvement opportunities.

* Trends in key metrics over time.
* Benchmark comparisons.
* Variation across team members.
* Variation across sectors.

The metrics guide improvement efforts.
