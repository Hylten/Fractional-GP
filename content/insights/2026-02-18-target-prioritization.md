Prioritization affects returns. Mathematical frameworks improve decisions. The firms that apply math outperform those that rely on intuition.

## The Scoring Frameworks

Target scoring uses multiple factors.

> Fit factors: Revenue, EBITDA, geography, sector.
> Motivation factors: Timing, urgency, owner situation.
> Competitive factors: Competition level, first-mover advantage.
> Return factors: Value potential, return potential, risk profile.

The scoring combines factors into priorities.

## The Allocation Optimization

Resource allocation follows prioritization.

* High-priority targets receive more resources.
* Low-priority targets receive minimal resources.
* Resource allocation maximizes expected return.
* Optimization considers constraints.

The allocation maximizes returns.

## The Dynamic Adjustment

Prioritization adjusts dynamically.

+ New information updates scores.
+ Market changes modify priorities.
* Competition shifts competitive position.
* Timing affects urgency.

The adjustment maintains accuracy.

## The Systematic Approach

Systematic approaches outperform intuition.

* Consistent scoring applies uniformly.
* Data-driven decisions reduce bias.
* Optimization finds global maximum.
* Documentation enables learning.

The approach improves over time.
